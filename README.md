## Le projet

![alt text](public/img/disney_cinema.jpg)
Disney Cinema est une collection de livres sur les films de Disney. Ma fille entame cette collection. L'application permet
de savoir ceux qu'elle désire et ceux qu'elle a déjà pour le communiquer aux proches qui veulent participer à cette collection.


<!-- GETTING STARTED -->
## Utilisation

3 actions possibles : voir la galerie, éditer la galerie, scrap la page web pour récupérer les données des livres.

### Voir la galerie

Pour les visiteurs souhaitant voir les livres désirés et déjà possédés.

  ```sh
  http://localhost/
  ```

### Editer la galerie

Pour permettre aux administrateurs de modifier la liste des livres.
   ```sh
   http://localhost/?edit
   ```

### Scraper la page
Les données sont issues du site d'Hachette. L'application scrape ce site pour mettre à jour sa base de données.
   ```sh
   http://localhost/scrap
   ```
